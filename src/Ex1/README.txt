To use this application:

1. Edit the Server.java run configuration to include the path to the directory you want to use as the server's root directory
2. Start the Server.java
3. Start the Client.java, any amount of clients can be started

List of commands:
"list" - lists all the files in the server
"get <filename>" - "downloads" the file from the server and saves it in the working directory as "receivedFile"
"exit" - exits the client
